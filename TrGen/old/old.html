Versions 0.8.1 and 0.8.2 of IXmaps Traceroute Generator are now available for download. Currently, only Windows XP and later are supported.

To install and run IXmaps Traceroute Generator version 0.8.2, you must be running as Administrator. The traceroute algorithm used usually completes within 10 seconds.

To install and run IXmaps Traceroute Generator version 0.8.1, no Administrator privileges are required. The Windows "tracert" program is used, and my take up to 6 minutes to do a single traceroute if no response is received from the destination host.

Other than the traceroute algorithm used, the two versions are identical. You may minimize the windows while the program is running.

When asked to do a full or custom install, pick the full install.

If there is temporarily high latency between your computer and the central database, you may get a "The connection to the central database is not working." pop-up. The TrGen client program waits 5 seconds after it sends a traceroute result to the server. If the confirmation is not received within that time, the pop-up will appear. Often, the data will have made it into the database.
