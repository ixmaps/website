IXmaps Traceroute Generator support page

IXmaps

The initial version of IXmaps Traceroute Generator is now available for download. Currently, only Windows XP and later are supported.

To install and run IXmaps Traceroute Generator, you must be running as Administrator.

When asked to do a full or custom install, pick the full install.

If there is temporarily high latency between your computer and the central database, you may get a "The connection to the central database is not working." pop-up. The TrGen cllient program waits 5 seconds after it sends a traceroute result to the server. If the confirmation is not received within that time, the pop-up will appear. Often, the data will have made it into the database.
